Manages custom variables for types not natively supported by the BOOSE library. Provides storage for real (floating-point) variables, arrays, and other extended types.

Remarks

This class uses a Dictionary to store variables by name, allowing flexible type storage that BOOSE's native variable system doesn't support. Primarily used for real number variables.

BOOSEapp-Documentation
Table of Contents

Namespace BOOSEapp

Classes

CanvasApp

Default BOOSE.ICanvas implementation backed by a Bitmap and Graphics. Provides simple drawing primitives (move, line, rectangle, circle, triangle, text) using the current pen/brush.

CircleApp

Handles the "circle" command to draw an unfilled circle at the current canvas position. Supports literal values, variables, and expressions for the radius parameter.

CommandFactoryApp

Application-specific command factory that constructs concrete drawing commands. Implements the Singleton pattern to ensure only one factory instance exists.

ElseApp

Handles the "else" command in if-else control structures. Provides alternate execution path when the preceding if condition evaluates to false.

EndApp
Form1

Main Windows Form application hosting the BOOSE interpreter with integrated editor and canvas preview. Provides a text editor for BOOSE programs and displays graphical output in real-time.

IfApp

Handles the "if" command for conditional execution in BOOSE programs. Evaluates a boolean condition and executes commands only when the condition is true.

IntApp

Handles the "int" command to create integer variables in BOOSE programs. Variables can be initialized with literal values, expressions, or default to 0.

IntValueParser

Static utility class for parsing integer values from strings, variables, or expressions. Provides unified parsing that checks BOOSE Program variables, CustomVariables, and evaluates expressions.

MoveToApp

Command that moves the canvas drawing cursor to a specified (x, y) coordinate without drawing.

PenApp

Class that sets the pen color using RGB values. Supports integer numbers, variables, and expressions for color components.

Program
RealApp

Handles the "real" command to create floating-point variables in BOOSE programs. Stores real variables in a custom VariableManager to bypass BOOSE library type restrictions.

RealValueParser

Parses real (floating-point) values from numbers, variable names, or expressions. Checks both BOOSE Program variables and CustomVariables storage.

RectApp

Handles the "rect" command to draw an unfilled rectangle at the current canvas position. Supports literal values, variables, and expressions for width and height parameters.

VarCommand

Handles variable assignment statements with numeric values and arithmetic expressions. Automatically creates new variables or updates existing ones in the StoredProgram.

VariableManager

Manages custom variables for types not natively supported by the BOOSE library. Provides storage for real (floating-point) variables, arrays, and other extended types.

WhileApp
WriteApp

Handles the "write" command to render text on the canvas at the current pen position. Supports literal strings, variables, expressions, and string concatenation.

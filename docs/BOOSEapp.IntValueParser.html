Remarks

This parser handles multiple input formats:

  • Literal integers: "42"
  • Variable names: "counter" (looks up in both BOOSE and custom storage)
  • Arithmetic expressions: "x + y * 2" (substitutes variables and evaluates)
  • Real variable names: Casts double values to integers Used throughout the application wherever integer values need to be parsed flexibly.

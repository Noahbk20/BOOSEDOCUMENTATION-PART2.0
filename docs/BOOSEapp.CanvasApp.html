Default BOOSE.ICanvas implementation backed by a Bitmap and Graphics. Provides simple drawing primitives (move, line, rectangle, circle, triangle, text) using the current pen/brush.

Remarks

  • Coordinates are in pixels, origin is the top-left of the canvas, Y increases downward.
  • The current drawing position is tracked by Xpos and Ypos.
  • Color is controlled via SetColour(int, int, int).
  • To resize/recreate the canvas, call Set(int, int).

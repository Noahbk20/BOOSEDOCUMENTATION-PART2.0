Examples

circle 50 // Draws circle with radius 50 pixels circle radius // Uses variable for radius circle size*2 // Uses expression for radius

Remarks

The circle is drawn centered at the current (Xpos, Ypos) coordinates using the current pen color. The radius parameter is parsed using IntValueParser to support literals, variables, and expressions. Negative radius values are rejected with an exception. Unlike the base CommandOneParameter.Execute(), this implementation bypasses BOOSE's evaluator to support custom variables stored in CustomVariables.

Remarks

Does not call base.Execute() because the base implementation uses BOOSE's evaluator which doesn't handle custom variables. Instead, uses IntValueParser directly.
